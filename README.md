## Personality Growth

技术之外，亦有广阔天空

积极性、对技术的热爱、…… 也许会随着时间，日益衰减，偶尔也需要一些鸡汤。

面对挫折、面对压力…… 你又会如何？

白头发是不是多起来了？是不是又长胖了？是不是很久没运动了？

以及，你是不是还是单身？

当然这些问题，在这里不会探讨，我也相信网上有足够多的资源可以参阅。

只是想说，首先是一个正常人，然后才是一个程序员。
